This is an excellent tip and something for anyone doing CSS styling to keep in mind: the html and body tags can be stylized. They may be optional in HTML5, but that does not mean that you shouldn’t use them. Take advantage of these semantic tags for styling so that you can cut down on the use of non-semantic div tags.
